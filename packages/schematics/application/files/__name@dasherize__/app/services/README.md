### 功能特性模块

services做为子应用需要用到的外部服务模块，子应用的所有需要用到的外部服务在此模块中。

> 服务通过脚手架生成，参考脚手架文档

#### 目录结构

目录结构概略图：

```

├── services                                          # 外部服务模块
│   ├── feature-1                                     # 服务1
│   ├── feature-2                                     # 服务2
│   ├── ...                                           # 服务...
│   ├── feature-n                                     # 服务n
│   └── public_api.module.ts                          # 公共api

```
