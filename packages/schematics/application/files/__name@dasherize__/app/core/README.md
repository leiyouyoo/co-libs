### Core模块

core做为子应用的核心模块，提供多语言及启动服务等核心服务。


##### 目录结构

目录结构概略图：

```
 ├── core                                # 核心模块
 │   ├── i18n
 │   │   └── i18n.service.ts             # 多语言服务
 │   ├── startup
 │   │   └── startup.service.ts          # 初始化项目配置
 │   └── core.module.ts                  # 核心模块文件

```
