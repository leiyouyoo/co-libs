## 写在前面

```bash
ng g co-cli:swagger --name xxx --url http://192.168.1.5:8002/swagger/xxx/swagger.json
```

## 命令格式

```bash
ng g co-cli:swagger --name xxx --url http://192.168.1.5:8002/swagger/xxx/swagger.json
```

## 额外参数

| 参数名               | 描述                                                  |
| -------------------  | ----------------------------------------------------- |
| `--name`             | swagger名称                                      |
| `--url`              |  swagger地址                                    |
