## 写在前面

```bash
 ng g co-cli:modules --name xxx
```

## 命令格式

```bash
 ng g co-cli:modules --name xxx
```

## 额外参数

| 参数名               | 描述                                                   |
| -------------------  | ----------------------------------------------------- |
| `--name`             | 命名文件名称                                           |
| `--path`             | 指定生成目录                                           |
