工具栏组件

## API

### toolbar

| 成员 | 说明 | 类型 | 默认值 | 全局配置 |
|----|----|----|-----|------|
| `[coAlign]` | 内容是否靠右对齐 | `boolean` | false |
| `[coMarginRight]` | 右间距 | `number` | 12 |
| `[coWidth]` | 宽度 | `string | number` | 'auto' |

### toolbar-item

| 成员 | 说明 | 类型 | 默认值 | 全局配置 |
|----|----|----|-----|------|
| `[coAlign]` | 是否单独靠右对齐 | `boolean` | false |
| `[coWidth]` | 单独指定宽度 | `number` | - |
| `[coMarginRight]` | 单独指定间距 | `number` | - |
